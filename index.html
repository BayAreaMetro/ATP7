MTC Active Transportation Program Cycle 5 Application Submission Hub

Welcome to the ATP Submission Hub, please follow the instructions in the widget below.

You may upload your application documents individually, or all at once as a .zip folder. Please submit the Statewide and Regional applications in .pdf format and the Project Programming Request (PPR) form as an Excel file. In the "File Description" text-box, please include the Project Title, the Applicant Agency, and the total number of files uploaded.

All documents must be received by MTC no later than June 15, 2020.

If you have any questions, contact Karl Anderson at (415) 778-6645 or kanderson@bayareametro.gov.
